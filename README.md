# Anomalib-Trainer

A simple Gradio interface for training [Anomalib](https://github.com/openvinotoolkit/anomalib) models.

<!-- ![Demo](assets/demo.png) -->

## Usage

1. Install dependencies

    ````bash
    pip3 install -r requirements.txt
    ````

2. Run the app

    ````bash
    python3 app.py
    ````
